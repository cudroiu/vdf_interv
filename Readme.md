# Run the code
The code can be run either with Python 2 or Python 3.

```
git clone https://github.com/cudroiu/vdf_interv.git
cd vdf_interv
python main.py
```

# Run unit tests
From the main folder ```vdf_interv``` run:
```
python -m unittest tests.sanity
```
